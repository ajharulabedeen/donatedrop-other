                                                      Dhiraj Ray bismaili 10 months ago

                                                      That is version issue with spring data. Spring data repository actually creates the implementation class based on the method defined. Hence, for the method, findOne(), it assumes that one is a model class attribute but it actually it's not. In the older version, findOne() actually looks DB by id only by default and that implmentation is now changed to findById() which also does the same thing.
